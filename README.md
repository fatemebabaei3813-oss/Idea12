# Idea12
ایده شماره یازدهم:
یه ایده دارم برای درست درس خوندن
تا الان ما با دید اینکه درس یک چیز سخته و شروع کردنش هم سخته داشتیم میرفتیم جلو
حالا بیاید جور دیگه بهش فکر کنیم
چون هر چیزی که ما انجام میدیم ساخته و پرداخته شده ذهنمونه
و وقتی من توی ذهنم از چیزی یک تعریف خوبی داشته باشم اون در واقعیت هم خوب دیده میشه
بیاید یادگیری رو انقدری جذاب کنیم واسه خودمون که ذوق داشته باشیم برای اینکه دوباره کی بشینم پای درس
اینجوری ذهنتم راحتر اونو میپزیره و راحتر میتونی مطالب رو بفهمی
ارزش یک بار امتحان رو که داره ، نداره؟
